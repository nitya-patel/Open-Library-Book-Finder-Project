# Open-Library-Book-Finder-Project

📚 Open Library Book Finder

📌 Project Overview

Open Library Book Finder is a web application that allows users to search for books and explore detailed information such as title, author, and publish year. The application fetches real-time data from the Open Library API and presents it in a clean, user-friendly interface.

This project demonstrates the use of JavaScript, API integration, and array operations like filtering, sorting, and searching.

🚀 Features

🔍 Search books by title or author
📖 Display book details (cover image, author name, publish year)
🔽 Filter books based on publish year or author
🔼 Sort books by publish year or edition count
⚠️ Handle missing data (fallback for unavailable images)
🎨 Responsive and visually appealing UI

🌐 API Used

This project uses the Open Library API to fetch book data.
API Endpoint Example:
https://openlibrary.org/developers/api

🛠️ Technologies Used

HTML
CSS (or Tailwind CSS / Bootstrap)
JavaScript (Fetch API)
Array Higher Order Functions (map, filter, sort)

🎯 Project Objective

The goal of this project is to build a functional and visually appealing web application that demonstrates:

API integration using fetch
Data manipulation using JavaScript
UI/UX design using CSS
